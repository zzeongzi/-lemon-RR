🔫 레몬 러시안룰렛 – 비트코인으로 즐기는 디스코드 기반 러시안룰렛 게임
레몬 러시안룰렛(LEMON Russian Roulette) 은

비트코인·라이트닝 네트워크를 활용해 디스코드에서 즐길 수 있는 고위험·고수익 러시안룰렛 배팅 게임입니다.

단순한 “재미용 봇”을 넘어서:

⚡ 라이트닝 네트워크로 실제 BTC 입·출금
🔍 투명한 기록, 공정한 룰
🧪 커뮤니티와 함께 진화하는 게임 실험실
을 목표로 하고 있습니다.

🕹 어떤 게임인가요?
레몬 러시안룰렛은 디스코드에서 명령어만으로 즐기는 텍스트 기반 배팅 게임입니다.

플레이어들은 각자 BTC(Sats) 를 예치하고 참여합니다.
라운드마다 총알이 들어갈 칸 수, 참가 인원, 배팅 금액 등이 정해집니다.
디스코드 봇이 난수를 사용해 “누가 생존하고, 누가 탈락하는지”를 결정합니다.
생존자는 상금을 나눠 갖고, 탈락자는 배팅 금액을 잃게 됩니다.
게임 구조는 단순하지만,

“얼마나 리스크를 감수할 것인가?”를 가지고 전략과 심리가 갈리는 것이 핵심 재미입니다.

⚡ 비트코인 & 라이트닝 연동
레몬 러시안룰렛은 라이트닝 네트워크를 기반으로 동작합니다.

/deposit 명령어로 라이트닝 인보이스를 발급받아 즉시 입금
/withdraw 명령어로 외부 라이트닝 지갑으로 바로 출금
서버 내부에서는 Sats 단위 잔액으로 게임에 참여
모든 입·출금은 Blink API + Discord 봇으로 자동 처리되며,

사용자는 디스코드에서 자신의 잔액과 게임 결과를 바로 확인할 수 있습니다.

🔒 공정성 & 투명성
현재 러시안룰렛 게임은 봇 코드에 따라 난수로 처리되며,

LEMON LOTTO에서 사용하는 비트코인 블록 해시 기반 Provably Fair 방식과의 연동도 고려 중입니다.

지향하는 방향은:

누구나 룰을 이해할 수 있고
결과 계산 과정을 검증 가능하게 만들고
운영자의 자의적인 조작이 필요 없는 구조
입니다.

커뮤니티 피드백에 따라:

게임 로그 공개 범위
난수 발생 방식(블록 해시 기반 등)
수수료 구조 / 상금 분배 방식
도 계속 조정해 나갈 계획입니다.

👥 커뮤니티에서 할 수 있는 것들
레몬 러시안룰렛 디스코드 서버에서는 단순히 게임만 하는 것이 아니라,

새 게임 모드/룰 제안
수수료/배당 구조 피드백
버그 리포트 및 기능 제안
비트코인/라이트닝 관련 정보 공유
등을 함께 할 수 있습니다.

러시안룰렛 외에도, 같은 레몬 시리즈로:

🍋 LEMON LOTTO (비트코인 블록 해시 기반 로또)
🐎 레몬 경마(가상 레이스)
같은 라이트닝/블록체인 연동 게임들을 확장해 나갈 예정입니다.

🏁 이런 분들께 추천합니다
비트코인/라이트닝 네트워크를 직접 써보면서 놀고 싶은 사람
디스코드 봇 게임, 텍스트 기반 파티 게임을 좋아하는 사람
“운과 배팅”이 만드는 심리전을 즐기는 사람
소규모, 실험적인 온체인/오프체인 프로젝트에 관심 있는 개발자/기획자
🔗 참여 방법
디스코드 서버에 접속
안내 채널에서 기본 규칙을 확인
/deposit으로 소액 Sats 입금
준비된 러시안룰렛 방에 들어가서 게임 명령어로 참여
